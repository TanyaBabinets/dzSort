Press Ctrl+E to Edit text

Press Ctrl+S to Save text

Создать html-страницу для отображения/редактирования текста. При открытии html-страницы текст отображается с помощью тега div. При нажатии Ctrl + E вместо блока div появляется многострочное текстовое поле textarea с тем же текстом, который теперь можно редактировать. При нажатии Ctrl + S вместо textarea появляется div с уже измененным текстом. Не забудьте выключить поведение по умолчанию для этих сочетаний клавиш.
